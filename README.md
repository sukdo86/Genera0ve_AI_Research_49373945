# Generative AI Research Project

This repository contains research materials for a project about the use of Artificial Intelligence in education. The project explores how students use AI tools, the possible benefits of AI in learning, and the challenges related to academic integrity, privacy, and responsible use.

## Project Structure

- `Literature Review`: stores journal articles, conference papers, books, and newspaper articles related to the research topic.
- `Quantitative Analysis`: stores survey data, survey questions, analysis files, Python scripts, CSV files, Excel files, and survey analysis reports.
- `Qualitative Analysis`: stores interview transcripts, interview protocols, consent forms, qualitative insights reports, and data visualisations.
- `Draft and Reports`: stores draft research proposals, conference papers, and final reports.
- `Additional Materials`: stores information sheets, photos, figures, and other media files.

## File Naming Convention

Files should be named clearly and consistently so that team members can easily understand the content, date, and version of each file.

Recommended format:

`YYYY-MM-DD_FileDescription_Version`

## Access Control

Sensitive files, such as raw survey data, interview transcripts, consent forms, and participant photos, should only be accessed by authorised team members. This helps protect participant privacy and supports ethical research data management.

## Collaboration Guidelines

Team members should use clear commit messages when uploading or editing files. Before making changes, contributors should check the latest version of the repository to avoid using outdated files. Major changes should be reviewed before being merged into the main branch.
